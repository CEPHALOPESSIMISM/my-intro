# ᴇᴢᴇᴋɪᴇʟ / ꜱʜᴀᴋᴇ 

ᴡᴀᴢᴢᴀᴘᴘᴘ ɪ'ᴍ ᴢᴇᴋᴇ ʙ) ᴍʏ ᴘʀᴏɴᴏᴜɴꜱ (ʏᴇꜱ ɪ ʜᴀᴠᴇ ᴛʜᴏꜱᴇ. ɪ ᴋɴᴏᴡ. ɪᴛ'ꜱ ʜᴏʀʀɪʙʟᴇ.) ᴀʀᴇ ɪᴛ/ɪᴛꜱ!!! ʙᴜᴛ ʜᴇ/ʜɪᴍ ɪꜱ ꜰɪɴᴇ, ᴛᴏᴏ.

ɪ'ᴍ ᴀɢᴇɴᴅᴇʀ ᴀɴᴅ ʙɪ (ᴏᴏʜ ʟᴀ ʟᴀ~) /ʀᴇꜰ

ɪ'ᴍ ᴀ ʏᴜᴍᴇꜱʜɪᴘᴘᴇʀ (ᴄʀɪɴɢᴇ!1!11!) ᴀɴᴅ ꜰɪᴄᴛɪᴏɴᴋɪɴ (ʟᴀᴍᴇ!11!1!!) ᴀɴᴅ ᴀ ᴛʜᴇʀɪᴀɴ (ʟᴏꜱᴇʀ!1!1!1!!)

ɪ'ᴍ ꜱᴄᴇɴᴇ ᴀɴᴅ ᴇᴍᴏ ᴀɴᴅ ɴᴜ-ᴍᴇᴛᴀʟʜᴇᴀᴅ ᴀɴᴅ ɪ ʟɪᴋᴇ ᴛᴀᴋɪɴɢ ɪɴꜰʟᴜᴇɴᴄᴇꜱ ꜰʀᴏᴍ ʜᴀʀᴅᴄᴏʀᴇ ᴇᴅᴍ, ʙʀᴇᴀᴋᴄᴏʀᴇ, ꜱᴘᴇᴇᴅᴄᴏʀᴇ, ᴅᴜʙꜱᴛᴇᴘ, ᴀɴᴅ ʀᴀᴠᴇ.
